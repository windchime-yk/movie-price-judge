# Movie price judge
近くにある映画館のポイントサービスを利用して安く観るとき、2ポイントでの減額と6ポイントでの無料のどっちが得なのか知りたかった。  
それだけ。

## コマンド
``` bash
deno run mod.ts <count>
```

## 出力例
``` bash
deno run mod.ts 30
```
